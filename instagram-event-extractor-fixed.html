⚠️ Setup Required

Before using this tool, you need to add your Anthropic API key to the code.

  1. Open this HTML file in a text editor (like Notepad or TextEdit)
  2. Find the line that says: const API_KEY = 'YOUR_API_KEY_HERE';
  3. Replace YOUR_API_KEY_HERE with your actual API key
  4. Save the file and refresh this page

Note: Never share your API key publicly or commit it to version control!
